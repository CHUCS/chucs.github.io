Apollo Server V3 - GraphQL + Node.js (part 1)

發表於 2022-02-21 分類於 Apollo Server - GraphQL Disqus：

前言

Apollo Server

一個易於使用、高度可擴展的開源框架，用於將服務和數據源連接到圖形。

優點：

  • 成熟的社群
  • Query 執行狀況和紀錄
  • 效能追蹤

GraphQL

GraphQL 是一種 API 的查詢語言。且在的 API 中的數據提供了完整且易於理解的描述，使開發者能夠準確詢問他們需要什麼。

優點：

  • 資料只拿剛好且彈性十足
  • 透過資料之間的關係連接 ，大幅減少來回 request 的次數
  • 前後端溝通成本減少
  • 前端不再被架構設計綁住，加速開發
